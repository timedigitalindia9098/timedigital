Frequently Asked Questions (FAQ)

Why are my formatting tags turning into empty blocks?

This usually means the operating system on the viewer's device is outdated. Modern updates across iOS, Android, and desktop ecosystems natively support unicode conversion libraries perfectly.

Can styled fonts be used in Bluesky profiles, bios, and display names?

Absolutely! You can copy and paste bold, italic, or custom font variations into your primary display name field and main bio description sections. However, your core username account handle (the text trailing the '@' character) must keep standard alphanumeric ASCII characters.

Does running text through this utility impact data privacy?

Not at all. This micro-tool operates completely serverless and client-side. The entire data processing matrix executes within your active local web page instance without storing string logs or transmitting telemetry details over network channels.
